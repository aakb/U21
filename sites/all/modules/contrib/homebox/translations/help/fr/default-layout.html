Depuis Administrer > Construction du site > Home box, cliquez sur mise en page en face du nom d'une page. Une page similaire à la page d'administration des blocs s'affiche.
Comme vous pouvez le voir, par défaut, 3 colonnes sont activées, elles sont toutes vides. La zone de blocs désactivés liste tous les blocs présents dans votre installation de Drupal. Vous pouvez assigner les blocs à n'importe quelle colonne par glisser/déposer. Vous pouvez réaliser la même opération en utilisant les listes de sélection. De cette manière vous pouvez créer la mise en page par défaut de votre page.
IMPORTANT : Vous devez comprendre qu'il ne s'agit que d'une mise en page par défaut. Une home box est conçue pour l'utilisateur puisse construire sa propre page, dans cette étape nous construisons donc la mise en page que l'utilisateur verra lors de sa première visite. Nous espérons que l'utilisateur modifiera cette page (déplacer des blocs, désactiver des blocs, etc.). Donc garder à l'esprit qu'il ne sagit que d'une mise en page par défaut.
En décochant les options Visible, Ouvert, Déplaçable, vous pouvez empêcher un bloc de s'afficher par défaut, d'être ouvert par défaut ou déplacé dans une autre colonne. La plupart du temps vous ne modifierez pas ces options. Lorsque vous avez terminé, vous pouvez cliquer sur le bouton Enregistrer les blocs. Vous pouvez vous rendre dans la section paramètres.
