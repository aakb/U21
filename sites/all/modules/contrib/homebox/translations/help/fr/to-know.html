Il y a plusieurs à savoir lorsque l'on utiliser Home box :
  • Le module jQuery UI est un enfer à faire fonctionner. Si vous rencontrez des problèmes, merci de consulter cette demande de support sur la page du projet Home box.
  • Si vous obtenez une page 404 Page non trouvée, cela peut signifier qu'aucun bloc n'est assigné à l'une des colonnes sur cette page. Rendez-vous dans la page de mise en page par défaut pour assigner quelques blocs.
  • Pourquoi ne peut-on pas exposer les nombre d'éléments d'une Vue (Views 2) ? Lire ce sujet.
  • Des problèmes avec vos blocs construis à partir de Vues (Views), lisez ce sujet Utilisez Views.
  • Les utilsateurs anonymes ne devraient pas avoir accès aux home box, on ne peut sauvegarder leurs préférences !
