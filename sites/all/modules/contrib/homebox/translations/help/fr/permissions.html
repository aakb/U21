Lorsque vous avez créé une page, vous devez spécifier les permissions qui y sont associées.
Depuis Administrer > Gestion des utilisateurs > Droits d'accès', sélectionnez tous les rôles qui peuvent accéder à votre home box.
Pour ce faire, cocher les case à cocher qui commence par "accéder à la homebox [homebox_name]" pour chacune des rôles.
Ne cochez jamais la case du rôle Anonyme, puisque Homebox n'est pas capable de sauvegarder les préférences utilisateurs d'un internaute qui n'existe pas sur votre site Drupal. Un message sera affiché incitant les internautes à s'identifier ou s'inscrire.
