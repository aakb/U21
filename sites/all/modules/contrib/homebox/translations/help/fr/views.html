Home box prend toute son ampleur lorsqu'il est utilisé avec Views 2. Views 2 vous permet de créer des vues dynamiques (des listes principalement) de votre contenu. Ces vues peuvent être manipulées par les utilisateurs avec : des filtres, un système de pagination, etc. tout ceci en Ajax.
Vous pouvez consulter directement le résumé si vous avez déjà lu cette section.
Afin d'avoir une page home box réellement interactive, il y a un certain nombre de paramètres à configurer dans votre vue. Dans cet exemple, nous allons utiliser une vue qui liste du contenu publié.
Tout d'abord, jetons un oeil à ce que nous voulons obtenir. Dans notre vue, la première dont nous avons besoin est affichage en bloc.
Dans les paramètres généraux assurez-vous que les options suivantes sont configurées. Vous pouvez comme à votre habitude configurer les autres options (style, champs, etc.).
Nous avons maintenant besoin d'exposer nos filtres. Ajoutons Type de noeud dans la section filtres. Assurez-vous de cliquez sur le bouton Exposer, comme dans la capture ci-dessous. N'oubliez pas les paramètres suivants dans les propriétés de votre filtre exposé. Avec tout ceci en place, vous obtenez un bloc entièrement en Ajax que vous pouvez utiliser dans l'une de vos Home box.

Résumé

Vous devez vérifier que votre vue :
  • a un titre
  • utilise Ajax
  • a une pagination
  • a ses blocs de filtres exposé dans des blocs (OUI)
  • a une valeur de Texte vide
  • a un nom d'administration (ce n'est pas obligatoire, mais plus pratique)
  • expose quelques filtres
  • que vous filtres aient l'option mémoriser (remember) cochée
