Rendez-vous dans Administrer > Construction du site > Home box. Dans le formulaire comme ci-dessous, entrez un nom dans le champ. Puis cliquez sur Ajouter une page.

Le formulaire se recharge avec votre nouvelle page listée. Il est conseillé de vous rendre la page édition. Allons-y.

A partir d'ici vous pouvez modifier ou supprimer une page. Mais ce que nous voulons ici, c'est créer un alias d'URL pour cette page. Sinon, elle ne sera accessible qu'à partie de l'adesse http://www.exemple.com/homebox/1, ce qui n'est pas très "propre".

Vous pouvez sauvegarder vos paramètres en cliquant sur le bouton Enregistrer la page et vous rendre dans la section mise en page par défaut.
