Depuis Administrer > Construction du site > Home box, cliquez sur paramètresen face de votre page. Un formulaire avec différents options s'affiche.
La première chose à paramètre est le nombre de colonnes que vous souhaitez pour votre home box.
Notez que so vous changez cette valeur, vous devez vous rendre sur la page de mise en page pour réassigner les blocs dans les nouvelle régions. Sinon l'utilisateur aura des colonnes vides.
Le paramètre suivant (et le seul pour le moment) vous permet d'autoriser ou non les utilisateurs à modifier les couleurs des blocs.
Pour leur donner cette possibilité, cochez l'option ci-dessous. Si cette fonction est activée, vous pouvez définir 6 couleurs personnalisées.
Utilisez le sélecteur de couleur (vous devez installer le module Colorpicker, sinon les champs ne seront que de type texte) pour définir les 6 couleurs. Lorsque c'est terminé, vous pouvez cliquez sur le bouton Enregistrer la configuration.
Mais s'il vous plaît ne partez pas nous n'avons pas terminé ! Rendez-vous maintenant dans la section permissions.
