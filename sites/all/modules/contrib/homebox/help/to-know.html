There's a couple of things to be aware of before using this module:
  • jQuery UI module can be evil to work properly. If you have troubles, please check this issue on Home box project page.
  • If you get a Page not found (404) error, this can mean that you have no blocks assigned to this home box page. So go to layout settings and give it some blocks.
  • Why can't we expose the number of items displayed by a View? Read this thread.
  • Trouble with your Views blocks and Home box, check Views topic.
  • Anonymous user shouldn't have access to Home box pages, since we can't save their preferences!
