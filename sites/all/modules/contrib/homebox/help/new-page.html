Go to Administer > Site building > Home box. In the form as shown below, type a name in the input. Then click Add page.

The form reloads with your newly created page listed. It is good pratice to visit the edit section. Let's go for it.

From here you can edit or delete your page. But what we want is to give an URL alias for that page. Otherwise, it will be only available through http://www.exemple.com/homebox/1, which isn't really user friendly.

You can save your change by clicking the Save page button and you can go to default layout topic.
