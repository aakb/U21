Home box allows your users to have a page that behave the same way as iGoogle, Netvibes or BBC.co.uk. You can think about Home box as MySite. But MySite is not going to be ported for Drupal 6. Home box will behave differently and will (for now) only be based on Drupal Block system. Which let's enough flexibility to serve any sort of content to the frontend user.
